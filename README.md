# 百度飞桨领航团零基础Python学习体会

python的特点
* 简洁性 实现同样的功能，python代码的行数往往是java的1/5。
* 易读性 代码像纯英语一样易于理解。
* 可扩展性 开源，任何人都可以做出自己的贡献。
